对象的自身引用（Self-Reference）

class A {
int x;
static A &a;

public :
A():a(*this){}
A(int x, A& aa):x(x),a(aa){}
};


void main()
{

A a1;
A a2(1, a1);

}
类引用自身须是static方式，目的取消类的基类独立性，禁止派生类。
典型的设计有Time类。
非静态引用没有实际意义。
